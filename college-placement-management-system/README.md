# 🚀 Placement Tracker System (MERN Stack)
A full-stack web application designed to streamline the college placement process by enabling students to track job applications, manage resumes, and stay updated with recruitment progress.


## 📌 Overview
The Placement Tracker System is built to solve the common problem students face during placement season — managing multiple applications, tracking interview rounds, and organizing resumes efficiently.

This application provides a centralized dashboard where students can monitor their placement journey while administrators can manage job postings and applicants.


## ✨ Features
### 👨‍🎓 Student Panel
* User authentication (Signup/Login)
* Track job applications and statuses
* Upload and manage resumes
* View upcoming interview schedules
* Personalized dashboard

### 🧑‍💼 Admin Panel
* Add and manage job opportunities
* View student applications
* Schedule interviews
* Manage placement data


## 🛠️ Tech Stack
### Frontend
* React.js
* CSS / Tailwind (if used)

### Backend
* Node.js
* Express.js

### Database
* MongoDB (Mongoose)


## 🔄 System Workflow
1. User logs in / registers
2. Student applies for job postings
3. Admin reviews applications
4. Interview schedules are updated
5. Students track status in dashboard


## ⚙️ Installation & Setup
### 1. Clone the repository
```bash
git clone https://github.com/Akanksha-2006/your-repo-name.git
cd your-repo-name
```

### 2. Install dependencies
#### Frontend
```bash
cd client
npm install
npm start
```

#### Backend
```bash
cd server
npm install
npm run dev
```

### 3. Environment Variables
Create a `.env` file in the server folder and add:

```
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
